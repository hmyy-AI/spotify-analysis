# spotify-analysis
# Project Overview
This project explores a comprehensive Spotify dataset containing over 100,000 tracks to uncover trends in music evolution from 1922 to the present. By leveraging Python's data science stack, I analyzed how song characteristics like duration, energy, and popularity have shifted over the decades.

# Tech Stack
Language: Python

Libraries: Pandas (Data Manipulation), Matplotlib & Seaborn (Data Visualization), NumPy.

Environment: Jupyter Notebook 
# Key Insights & Findings
Performed Exploratory Data Analysis (EDA) on a dataset of 100k+ songs to identify industry trends.

Discovered a strong correlation (0.76) between Energy and Loudness, providing insights into modern music production.

Analyzed the decline in song duration post-2010, linking it to the rise of social media and streaming algorithms.
